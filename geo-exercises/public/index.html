Express Game API

api/game/nearbyPlayers

The endpoint can be used for getting all nearby players, excluding yourself.
    
      {
        "username": "username here",
        "password": "password here",
        "lon": "longitude as a number here",
        "lat": "latitude as a number here",
        "distance": "distance (meters) in which to search for other players"
      }

api/game/getPostIfReached

Returns a Post if it's within range
    
      {
        "postID": "Post's id here",
        "lon": "longitude as a number here",
        "lat": "latitude as a number here"
      }

api/game/updatePosition

The endpoint can be used to update your player position
    
      {
        "username": "username here",
        "password": "password here",
        "lon": "longitude as a number here",
        "lat": "latitude as a number here",
        "distance": "distance (meters) in which to search for other players"
      }
